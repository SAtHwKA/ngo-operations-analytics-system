# 🌿 NGO Operations Analytics System

## 📌 Overview

This project is a Streamlit-based dashboard that helps NGOs analyze their operational data and improve planning strategies.

---

## 🎯 Features

* Upload dataset
* View data analysis metrics
* Visualize insights using graphs
* Smart planning tool
* Plan improvement suggestions

---

## 📊 Visualizations

* Success Rate by Activity
* Budget Distribution by Region
* Budget vs Success Rate
* Priority Level Distribution

---

## 🧠 Smart Planning Tool

* Takes user inputs:

  * Team size
  * Resources
  * Budget

* Calculates performance score

* Classifies plan as:

  * Efficient ✅
  * Moderate ⚠
  * Inefficient ❌

* Suggests better plans if needed

## ▶️ How to Run

1. Install dependencies:

```bash
pip install streamlit pandas matplotlib
```

2. Run the project:

```bash
streamlit run app.py
```

3. Open in browser:
   http://localhost:8501
