# 🌿 NGO Operations Analytics System

## 📌 Overview

This project is a Streamlit-based analytics system designed to evaluate NGO operations and provide insights for improving efficiency, planning, and resource utilization.

---

## 🎯 Features

* Dataset upload
* Data analysis metrics
* Graph visualizations
* Smart planning tool
* Plan optimization suggestions

---

## 📂 Project Structure

```bash
app.py
planner_data.csv
README.md
```

---

## ▶️ How to Run

### 1. Install dependencies

```bash
pip install streamlit pandas matplotlib
```

### 2. Run the application

```bash
streamlit run app.py
```

### 3. Open in browser

```
http://localhost:8501
```
